# Папки,файлы и их предназночение
Использовать описания только при работе с конкретным файлом

## Корень проекта
  main.py - запускающий файл
  make_backup.py - бекэнд проекта
  requirements.txt - зависимости
  .env - секреты
  .gitignore - гит игнор

## CORE
  app\core\theme\dark_theme.py - Темная тема для всего проекта 
  app\core\logging\logger_config.py - Логирование всего проекта

## SERVICES
  app\services\build_oi_summary.py - получение данных из бд для передачи в oi_page.py
  app\services\load_logs_to_ui.py - подготовка файла app.log, для передачи в logs_page.py
