# Папки,файлы и их предназночение
Использовать описания только при работе с конкретным файлом

## Корень проекта
  main.py - запускающий файл
  make_backup.py - бекэнд проекта
  requirements.txt - зависимости
  .env - секреты
  .gitignore - гит игнор

## CORE
  app\core\theme\dark_theme.py - Темная тема для всего проекта 
  app\core\logging\logger_config.py - Логирование всего проекта

## SERVICES
  app\services\build_oi_summary.py - получение данных из бд для передачи в oi_page.py
  app\services\load_logs_to_ui.py - подготовка файла app.log, для передачи в logs_page.py
  app\services\build_oi_analytics.py - подготовка данных для таблицы в analytics_page.py
  app/services/sync_price_history.py — определяет недостающие периоды и догружает цены по всем тикерам через API в БД

## REPOSITORY
  app\repository\oi_repository.py - берем данные из таблицы futures_oi, передаем в build_oi_analytics.py
  app/repository/price_repository.py — читает и записывает данные цен в БД (paper_price), включая последнюю дату по тикеру

## API
  app/api/moex_client.py — делает HTTP-запросы к MOEX API и возвращает сырые данные свечей


## UI
  app\ui\windows\main_window.py - Главное окно 
  app\ui\sidebar\sidebar_widget.py - сайдбар меню
  app\ui\pages\analytics_page.py - страница анализа
  app\ui\pages\logs_page.py - страница логов
  app\ui\pages\oi_page.py - страница ОИ
  app\ui\pages\price_page.py - страница цены
  
## STORAGE
  storage\logs\app.log - логи со всего проекта
  storage\logs\error.log - логи ошибок со всего проекта
